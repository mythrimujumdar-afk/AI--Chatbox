# 📄 AI PDF Chatbot (RAG)

This project is an AI-powered chatbot that allows users to interact with PDF documents.

## 🚀 Features
- Upload PDF
- Ask questions
- Get AI-generated answers

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Ollama (Llama 3)
